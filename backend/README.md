A _Scala_ backend using _Akka-HTTP_ for Nabu.

## Test
```bash
sbt test
```

## Running locally
```bash
sbt run
```

## Autoformatting
```bash
sbt scalafmtAll
```

## Running Scapegoat
```bash
sbt scapegoat
```

## Building jar
```bash
sbt assembly
```
